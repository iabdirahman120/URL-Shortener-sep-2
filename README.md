# URL Shortener

Et URL shortener projekt bygget med React, Node.js og PostgreSQL.

## Live site

https://url-shortener-sep2.vercel.app

## Tech stack

- **Frontend:** React, Vite, Tailwind CSS, shadcn/ui
- **Backend:** Node.js, Express
- **Database:** PostgreSQL (Neon)
- **Hosting:** Vercel

## Funktioner

- Opret konto og log ind
- Forkort lange URLs
- Custom alias
- Udløbsdato på links
- Se og slet dine links
